How to Play Simon Game!!

1. To start the game Click on Play button and then on "Start the Game".

2. The game will show the first colour in the sequence e.g:(blue). The user clicks on the blue button.

3. The game will show the next colour e.g:(red). The user has to remember the sequence is blue, red and have to click first blue and then red, then the game continues.

4. If he/she forgets the sequence, then the game ends.

5. To restart, click on Game over.
